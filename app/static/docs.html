API Documentation

Getting Started

ToxidAPI provides a simple REST API for text analysis. Here's how to get started:

# Basic request example curl -X POST https://api.toxidapi.com/v2/analyze \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json" \ -d '{"text": "Your text here"}'

Response Format

The API returns JSON responses with toxicity scores and sentiment analysis:

{ "toxicity": { "score": 0.92, "is_toxic": true }, "sentiment": { "score": -0.75, "label": "NEGATIVE" } }

Authentication

All API requests require an API key to be included in the Authorization header. You can get your API key from the API Keys section.

Rate Limiting

API calls are rate limited based on your plan. Check the Rate Limits section for more details.

Error Handling

{ "error": { "code": "invalid_request", "message": "Invalid API key provided" } }
